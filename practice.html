LE WAGON

Portées par les évolutions technologiques, les entreprises du numérique innovent, inventant tous les jours de nouveaux produits. La formation a parfois un wagon de retard... Les jeunes diplômés, ayant pourtant une vraie soif d'apprendre, n'ont pas toujours le bagage technique suffisant pour intéresser des recruteurs ou se lancer sur des projets personnels.

A travers des formations complémentaires en programmation web, allant du cours d'initiation au programme immersif de 2 mois, le wagon veut permettre à ces jeunes talents de devenir des acteurs de l'innovation.

NOUS ENSEIGNONS

L'algorithmique dans le langage ruby & les techniques de growth hacking (scraping, interrogation d'APIs). Nous appliquons de manière pragmatique le code aux problèmes des entrepreneurs.
Les concepts objet et les design patterns classiques (MVC, Page-Controller), notions-clefs pour prendre en main n'importe quel nouveau framework web. Ruby on Rails nous permet de prototyper très vite n'importe quelle idée.
La modélisation de bases de données SQL, l'intégration de librairies open-source (fileupload, géolocalisation, authentification, paiement, Oauth, etc.) et la programmation côté navigateur avec JavaScript et AJAX.
Les best practices de developpement web (collaboration sur Github, versionning de code, déploiement continu sur Heroku, gestion des photos sur Amazon S3, envoi de mail avec SMTP externe comme Mandrill, etc.)
